## 快速运行

在前端的世界里，所码即所得。当我想要了解一个项目的时候，我会直接运行看效果。

接下来，假设你已经具备一定的前端工程化开发经验。

```bash
cd ~/Desktop
git clone https://github.com/leer0911/myAudio.git
cd myAudio
npm i
npm run dev
```
